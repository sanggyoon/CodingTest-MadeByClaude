# Q20260108 - 작업 스케줄러

## 문제 설명
컴퓨터 시스템에서 작업을 효율적으로 처리하기 위한 스케줄러를 만들려고 합니다.

각 작업은 요청 시간과 소요 시간을 가지고 있으며, 하나의 작업을 처리할 때는 다른 작업을 동시에 처리할 수 없습니다.

스케줄러는 다음 규칙으로 작업을 처리합니다:
1. 현재 시간에 처리 가능한 작업 중에서 **소요 시간이 가장 짧은 작업**을 먼저 처리합니다. (SJF: Shortest Job First)
2. 처리 가능한 작업이 없으면 다음 작업이 요청될 때까지 대기합니다.
3. 처리 중인 작업은 중단할 수 없습니다.

모든 작업의 평균 대기 시간을 최소화하려고 할 때, 평균 대기 시간을 반환하세요.

**대기 시간**: (작업이 완료된 시간 - 작업이 요청된 시간)

## 제약 사항
- 작업의 개수는 1개 이상 500개 이하입니다.
- 각 작업의 요청 시간(requestTime)은 0 이상 1,000 이하입니다.
- 각 작업의 소요 시간(duration)은 1 이상 1,000 이하입니다.
- 평균 대기 시간의 소수점 이하는 버립니다.

## 입출력 예시

### 예시 1
**입력:**
```java
int[][] jobs = {{0, 3}, {1, 9}, {2, 6}};
```

**출력:**
```
9
```

**설명:**
- 시간 0: 작업 A(소요 3) 시작 → 시간 3에 완료 (대기: 3-0=3)
- 시간 3: 작업 C(소요 6) 시작 → 시간 9에 완료 (대기: 9-2=7)
- 시간 9: 작업 B(소요 9) 시작 → 시간 18에 완료 (대기: 18-1=17)
- 평균 대기 시간: (3+7+17) / 3 = 9

### 예시 2
**입력:**
```java
int[][] jobs = {{0, 10}, {2, 3}, {9, 3}};
```

**출력:**
```
9
```

**설명:**
- 시간 0: 작업 A(소요 10) 시작 → 시간 10에 완료 (대기: 10-0=10)
- 시간 10: 작업 B(소요 3) 시작 → 시간 13에 완료 (대기: 13-2=11)
- 시간 13: 작업 C(소요 3) 시작 → 시간 16에 완료 (대기: 16-9=7)
- 평균 대기 시간: (10+11+7) / 3 = 9.333... → 9

### 예시 3
**입력:**
```java
int[][] jobs = {{0, 3}, {1, 9}, {500, 6}};
```

**출력:**
```
6
```

**설명:**
- 시간 0: 작업 A(소요 3) 시작 → 시간 3에 완료 (대기: 3-0=3)
- 시간 3: 작업 B(소요 9) 시작 → 시간 12에 완료 (대기: 12-1=11)
- 시간 500: 작업 C(소요 6) 시작 → 시간 506에 완료 (대기: 506-500=6)
- 평균 대기 시간: (3+11+6) / 3 = 6.666... → 6

## 힌트
<details>
<summary>힌트 1 - 자료구조</summary>

이 문제는 **우선순위 큐(PriorityQueue)**를 사용하면 효율적으로 해결할 수 있습니다.
- 처리 가능한 작업들 중에서 소요 시간이 가장 짧은 것을 자동으로 찾아줍니다.
- `PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> a[1] - b[1]);`

</details>

<details>
<summary>힌트 2 - 알고리즘 흐름</summary>

1. 먼저 jobs 배열을 요청 시간 기준으로 정렬합니다.
2. 현재 시간(currentTime)과 작업 인덱스(index)를 관리합니다.
3. 반복:
   - 현재 시간에 처리 가능한 작업들을 우선순위 큐에 추가
   - 큐에서 소요 시간이 가장 짧은 작업을 꺼내 처리
   - 현재 시간을 업데이트
   - 대기 시간을 누적
4. 총 대기 시간 / 작업 개수

</details>

<details>
<summary>힌트 3 - 엣지 케이스</summary>

- 처리 가능한 작업이 없을 때는 다음 작업의 요청 시간으로 점프해야 합니다.
- 우선순위 큐가 비어있고, 아직 처리할 작업이 남아있다면:
  ```java
  if (pq.isEmpty() && index < jobs.length) {
      currentTime = jobs[index][0];
  }
  ```

</details>
