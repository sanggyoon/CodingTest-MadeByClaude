# Q20260112 - 체육복 대여

## 문제 설명
어느 학교에서 체육수업을 하려고 합니다. 그런데 도난 사고로 일부 학생들이 체육복을 도난당했습니다. 
다행히 여벌의 체육복을 가져온 학생들이 있어서, 이 학생들이 체육복을 빌려줄 수 있습니다.

학생들의 번호는 체격 순으로 매겨져 있어, **바로 앞번호의 학생이나 바로 뒷번호의 학생에게만** 체육복을 빌려줄 수 있습니다.
예를 들어, 4번 학생은 3번 학생이나 5번 학생에게만 체육복을 빌려줄 수 있습니다.

체육복이 없으면 체육수업을 들을 수 없기 때문에, 최대한 많은 학생이 체육수업을 들을 수 있도록 해야 합니다.

## 제약사항
- 전체 학생의 수는 2명 이상 30명 이하입니다.
- 체육복을 도난당한 학생의 수는 1명 이상 n명 이하이고 중복되는 번호는 없습니다.
- 여벌의 체육복을 가져온 학생의 수는 1명 이상 n명 이하이고 중복되는 번호는 없습니다.
- 여벌 체육복을 가져온 학생이 도난당했을 수 있습니다. 이때 이 학생은 체육복을 하나만 도난당했다고 가정하며, 남은 체육복이 하나이기에 다른 학생에게 체육복을 빌려줄 수 없습니다.

## 입력 형식
```
n: 전체 학생의 수
lost: 체육복을 도난당한 학생들의 번호가 담긴 배열
reserve: 여벌의 체육복을 가져온 학생들의 번호가 담긴 배열
```

## 출력 형식
체육수업을 들을 수 있는 학생의 최댓값을 return 하도록 solution 메서드를 작성하세요.

## 입출력 예

### 예제 1
```java
입력: n = 5, lost = [2, 4], reserve = [1, 3, 5]
출력: 5
```
**설명:**
- 1번 학생이 2번 학생에게 체육복을 빌려주고
- 3번 학생이 4번 학생에게 체육복을 빌려주면
- 5명 모두 체육수업을 들을 수 있습니다.

### 예제 2
```java
입력: n = 5, lost = [2, 4], reserve = [3]
출력: 4
```
**설명:**
- 3번 학생이 2번 또는 4번 학생 중 한 명에게만 빌려줄 수 있습니다.
- 3번 학생이 2번이나 4번 중 누구에게 빌려주든 한 명은 체육복이 없어 4명만 수업을 들을 수 있습니다.

### 예제 3
```java
입력: n = 3, lost = [3], reserve = [1]
출력: 2
```
**설명:**
- 1번 학생은 3번 학생과 번호 차이가 2이므로 빌려줄 수 없습니다.
- 따라서 3번 학생은 체육수업을 들을 수 없어 2명만 들을 수 있습니다.

### 예제 4
```java
입력: n = 5, lost = [2, 3], reserve = [3, 4]
출력: 5
```
**설명:**
- 3번 학생은 도난당했지만 여벌이 있어서 자신의 체육복을 입을 수 있습니다.
- 4번 학생이 2번이나 3번 학생에게 빌려줄 수 있는데, 3번은 이미 자신의 여벌로 입었으므로 2번에게 빌려줘야 합니다.
- 하지만 4번과 2번은 번호 차이가 2이므로... 아니 잠깐! 4번이 3번에게 빌려줄 필요가 없으므로, 2번에게... 아니 이것도 차이가 2네요.
- 실제로는 3번 학생이 자기 여벌을 쓰고, 4번 학생이 3번에게 빌려줄 수 없으므로... 다시 생각하면:
  - 3번은 여벌이 있지만 도난당해서 자기 것 입음
  - 4번은 3번 또는 5번에게 빌려줄 수 있지만, 3번은 이미 있으니 필요 없음
  - 따라서 4번이 3번의 역할을 대신해서 2번에게 빌려줄 수 있을까? 아니요, 번호 차이가 1이 아닙니다.
- 올바른 해석:
  - 3번 학생은 도난당했지만 여벌이 있어 자기 것 착용 (수업 가능)
  - 4번 학생은 여벌이 있고, 3번은 이미 해결됐으니 3번 학생(왼쪽 방향)이 아니라... 음, 3번이 2번에게 빌려줄 수 있을까요?
  - 다시! 3번은 도난+여벌=1벌이므로 못 빌려줌. 4번이 3번(이미 있음)이나 5번에게 빌려줄 수 있음.
  - 결국 2번은 체육복이 없어서 4명만 가능? 하지만 정답은 5명...
  
**올바른 해석:**
- 3번: 도난당했지만 여벌 있음 → 자신이 입음 (다른 학생에게 빌려줄 수 없음) ✅
- 2번: 도난당함, 1번이나 3번에게 받아야 함 → 하지만 3번은 못 빌려줌
- 4번: 여벌 있음, 3번이나 5번에게 빌려줄 수 있음 → 하지만 3번은 필요 없음
- 결론: 2번 학생만 체육복이 없어서 4명만 가능

**주의: 이 예제는 실제로 4명이 정답일 수 있습니다. 문제를 풀면서 확인해주세요!**

<details>
<summary>힌트 2: 중요한 예외 상황은?</summary>

**여벌 체육복을 가져온 학생이 도난당한 경우**를 가장 먼저 처리해야 합니다!
- 이 학생은 자기 여벌로 자기가 입어야 하므로, 다른 학생에게 빌려줄 수 없습니다.
- lost와 reserve 배열에서 둘 다 제거해야 합니다.
</details>

<details>
<summary>힌트 3: 어떤 순서로 처리해야 할까요?</summary>

1. 여벌 체육복을 가진 학생이 도난당한 경우를 먼저 처리 (자기가 입음)
2. 학생 번호를 정렬
3. 도난당한 각 학생에 대해:
   - 앞 번호(-1) 학생이 여벌이 있으면 빌림
   - 없으면 뒷 번호(+1) 학생이 여벌이 있으면 빌림
4. 체육복을 입을 수 있는 학생 수 = 전체 학생 수 - 빌리지 못한 학생 수
</details>
