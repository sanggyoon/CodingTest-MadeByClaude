# Q20260109 - 동전 거스름돈 (그리디 알고리즘)

## 📝 문제 설명

편의점에서 손님에게 거스름돈을 줘야 합니다. 거스름돈을 줄 때는 **동전의 개수를 최소화**하여 주고 싶습니다.

현재 편의점에는 다음과 같은 동전들이 무한히 있습니다:

- 500원
- 100원
- 50원
- 10원

거스름돈 금액이 주어질 때, **최소 동전 개수**를 구하는 프로그램을 작성하세요.

---

## 📋 제약 사항

1. 거스름돈은 **10원 단위**로만 주어집니다 (항상 10으로 나누어떨어짐)
2. 거스름돈은 **0원 이상 100,000원 이하**입니다
3. 동전은 무한히 있다고 가정합니다
4. 동전 종류는 500원, 100원, 50원, 10원 4가지입니다

---

```java
int change  // 거스름돈 (원)
```

---

## 📤 출력 형식

```java
int result  // 최소 동전 개수
```

<details>
<summary>💭 힌트 2 - 어떻게 구현할까?</summary>

1. 큰 동전부터 차례대로 확인 (500 → 100 → 50 → 10)
2. 각 동전으로 **최대한 많이** 거슬러줄 수 있는 개수 계산
3. 사용한 동전 금액만큼 거스름돈에서 빼기
4. 다음 동전으로 이동

</details>
